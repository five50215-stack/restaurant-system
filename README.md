# 餐廳系統 — 網頁重製版

這是把你以前用 **Visual Basic 6** 寫的「餐廳系統」專題,1:1 重建成可以在 Windows 11 + 現代瀏覽器直接執行的網頁版。
原本的 VB6 程式依賴已停止支援的 Flash OCX、Jet 資料庫引擎和 32 位元執行環境,在新電腦上幾乎無法執行;這個版本保留了原汁原味的畫面、資料與操作邏輯。

## 怎麼執行

雙擊 **`啟動餐廳系統.bat`**(需要電腦有安裝 Python),瀏覽器會自動開啟 `http://localhost:8137/`。

或手動:在本資料夾開命令列,執行 `python -m http.server 8137`,再用瀏覽器開 `http://localhost:8137/`。

## 完整還原內容

- **進入點「第一個畫面」**:跑馬燈、時鐘、滑過圖片顯示介紹、消費者/廚師/管理者三個入口。
- **消費者點餐(系統選擇菜單)**:輸入人數與預算 → 依口味/分類/預算篩選 74 道菜 → 點餐扣預算 → 結帳扣庫存 → 訂位 → 顧客訂單。
- **套餐系統(套餐選擇)**:9 種套餐分類篩選、看內容與價錢、結帳扣原料。
- **廚師端**:登入(8137/8137)→ 新增菜單、新增套餐(可挑 10 道菜自動聚合原料、算 85 折價)。
- **管理者端**:登入(帳 8137 / 密 8137 / 驗證碼)→ 276 筆庫存管理:排序、查詢、過期提醒、剩餘天數、刪除、新增、字體/顏色設定。
- **意見反應、顧客意見顯示、井字遊戲(含電腦 AI)** 等全部移植。
- 所有圖片皆從原始 `.frx` 表單資源與專案 image 資料夾直接取出;選單/套餐/庫存資料皆從原始 `.mdb` 匯出。

## 帳號密碼

- 廚師登入:帳號 `8137`、密碼 `8137`
- 管理者登入:帳號 `8137`、密碼 `8137`、再輸入畫面上的隨機驗證碼

## 檔案結構

| 檔案 | 說明 |
|------|------|
| `index.html` | 入口頁 |
| `layout.json` | 25 個 VB 表單的版面(由 `.frm` 解析,twips 換算為像素) |
| `data.js` | 菜單 74 筆、套餐 9 筆、庫存 276 筆(由 `.mdb` 匯出) |
| `renderer.js` | VB6 控制項 → 網頁渲染器(座標、字型、顏色、z-order) |
| `app.js` | VB 事件邏輯移植 + MsgBox/InputBox/Timer/表單導覽 |
| `assets/img/` | 從 `.frx` 與專案 image 取出的圖片 |

> 註:原始 VB6 的 Flash 動畫(welcome.swf 等)在現代瀏覽器已無法原生播放,畫面以原圖呈現;其餘功能與版面皆與原程式一致。
